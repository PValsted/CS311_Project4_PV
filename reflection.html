  • In general, this project wasn't super challenging for me, but I did find I was super rusty on HTML and CSS. Because of this I suspect there are many things I could've done differently or more efficiently, and I need to refresh myself on it when I have more time. As for Bootstrap, it seems a bit complex and maybe a bit intimidating, but the flipside is I can see how impressive the amount of stuff you can do with it is. It makes a lot of things I remember doing with HTML a while back much simpler, and although it is nice to design everything manually, I enjoy the time saved and peace of mind using a framework gives me.
